JAZZ / CLASS 02

小学生・入門クラス

レッスンには、バレエの要素を取り入れながらダンスの基礎を養います。 テンポの速いものからスローな曲までどんな踊りも踊れるよう指導していきます。

JAZZ / CLASS 03

小中高生・初中級クラス

このクラスは、アップをただのストレッチと考えず、ダンスを踊るための基礎作りと考えます。 そのため前半はバーレッスンを取り入れ、後半にターンやステップを取り入れ最後にダンスを踊ります。

JAZZ / CLASS 04

一般クラス

大人のためのクラスです。10代～50代の方が楽しく汗をかきながらダンスを踊ります。 大人になってからダンスを始められた方が多いため、初心者の方でも安心してレッスンにご参加いただけます。
